Compile a ROM:

The taboo behind compiling ROMs has discoraged many users from learning, and existing guides are limited.
This (Guided) Script aims to teach users in an unconventional method, though it does begin semi-specific to exynos9810 the thorough detail on why and how each command is occuring should inspire users from other/future devices to test.
Nevertheless, the script is defaulted to compile the Unofficial LineageOS 15.1 that you can see on the 'Home' page, if users don't feel experimental.

Arch Linux Install:

Arch Linux is a saviour for running old hardware, almost giving your machine a new life.
Install the packages only you want, and not pointlessly losing resources from packages you don't even use.
The fear of it being too 'barebones' prevents people from installing it so this guide, from @synt4x93, should deliver you to a better starting point.
